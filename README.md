# 프로젝트 소개

- 서비스 특징 : Openvidu와 Openpose를 응용한 자세교정 기능을 갖춘 실시간 화상회의 스터디 플랫폼
- 주요 기능
    - 회원 관리
    - 스터디
    - 화상 회의
    - 자세 교정(AI)
    - 게시판
- 주요 기술
    - REST API
    - WebRTC
    - Openpose
- Framework
    - Frontend : React
    - Backend : Spring/Spring Boot
- Web Application

# 팀 소개
